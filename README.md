# Project Name : DigiTools Hub
# Description:
Digitoole is a web-based utility platform that provides multiple digital tools in one place. The main goal of this project is to simplify everyday online tasks—such as text processing and quick tool usage—through a clean and user-friendly interface.

# Key Features:
1. User can access different tools and buy this and use it for daily life & increase productivity.
2. The Interface is clean and easy to use , making it accessible for both beginners and expert users.
3. The DigiTools Website is very lightweight, ensuring fast loading times and a smooth experience on both mobile and desktop devices.
# Technology:
FrontEnd: HTML5, CSS3, Tailwind CSS
UI: DaisyUI
Programming language: Javascript
Library: React.js
Deployment: Netlify
